# Steel Assault

A lightweight browser tank battle game built with plain Canvas and zero build tooling.

## Controls

- `WASD` or arrow keys to move
- `Space` to fire
- `R` to restart; after clearing a wave, press `R` to enter the next one

## Objective

- Protect the base at the bottom of the map
- Destroy every enemy tank in the current wave
- Use breakable brick cover and solid steel walls to control the battlefield

## Run locally

Open `index.html` directly, or start any static file server in the project directory, for example:

```powershell
npx serve .
```

Then visit the printed local URL.
